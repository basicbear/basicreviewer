# Development
## Testing
- Run all tests: `uv run pytest -v`
- Run just init tests: `uv run pytest ./tests/init -v`
- pull tests: `uv run pytest ./tests/pull -v`

#### Unreviewed Tests
- cmd: pull - all except base
- cmd: extract - all
- cmd: sum - all
- cmd: export - all
- cmd: import - all
- cmd: mcp-serv - all
